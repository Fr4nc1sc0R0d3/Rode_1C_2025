Descripción detallada

Estructura que representa la configuración de un LED.

Contiene los parámetros necesarios para controlar un LED, incluyendo el modo de funcionamiento, el número de LED, la cantidad de ciclos de encendido/apagado y el periodo de cada ciclo.

◆ BCDToGPIO()

void BCDToGPIO ( int8_t  digitoBCD,
gpioConf_t *  vectorGPIO 
)

Cambia el estado de 4 pines GPIO según un dígito BCD.

Recorre los 4 bits menos significativos del dígito BCD, evaluando si están en 1 o 0, y enciende o apaga el pin correspondiente del arreglo vectorGPIO usando GPIOOn o GPIOOff.

El bit menos significativo (b0) se corresponde con el primer pin del vector (GPIO_20). Luego, se avanza bit a bit desplazando el número hacia la derecha.

Parámetros
digitoBCDDígito codificado en BCD (de 0 a 9).
vectorGPIOPuntero a un vector de estructuras gpioConf_t con la configuración de los pines GPIO.

void encenderLED ( int32_t  digito32Bits,

gpioConf_t *  vectorConfiguracionLED,

)

Muestra un número de hasta 3 dígitos en un display de 7 segmentos multiplexado.

Esta función convierte el número recibido a BCD, y luego lo muestra en los 3 displays mediante multiplexado. Cada display se activa uno a la vez, con un retardo corto para “latchear” los datos en el CD4543.

Parámetros
digito32BitsNúmero entero a mostrar (hasta 3 dígitos).
cantidadDigitosCantidad de dígitos que se desea mostrar (por ejemplo, 3).
vectorConfiguracionLEDVector con los pines D1-D4 conectados al BCD.
vectorSeleccionLEDVector con los pines SEL_1 a SEL_3 de selección del display.
